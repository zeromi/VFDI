# VFDI
 VirtualFloppyDiskImage，一个可以创建虚拟软盘，并可以对其进行读写的虚拟软盘编辑程序。
## 功能规划

    1.可生成1.44M虚拟软盘文件
    2.可将虚拟软盘格式化为FAT12格式
    3.可读取虚拟软盘文件
    4.可读取虚拟软盘目录和目录树
    5.可将文件写入虚拟软盘和创建目录

## 完成情况

    1.可生成1.44M虚拟软盘文件
    2.可将虚拟软盘格式化为FAT12格式
    3.可读取虚拟软盘文件
    4.可读取虚拟软盘目录和目录树，并可以切换目录
	5.可以输出虚拟软盘中的文件(不支持文件夹)
	6.提供了简易的控制台命令执行
	7.暂时不支持文件写入
